# 🎨 Public Portfolio (`/portfolio`)

## Overview
This directory serves as the public-facing networking asset encompassing case studies, career roadmaps, and resume landing pages designed specifically for recruiters and hiring managers.

## Components & File Structure
- **`index.html`**: The hero landing page presenting the core pitch—who I am, the value I bring, and my primary skill orientations spanning AI PM frameworks and generalized B2B Enterprise dynamics.
- **`case-study-1.html` & `case-study-2.html`**: Deep dive validations of my career achievements designed distinctly to showcase problem, constraints, methodology, and empirical metrics.
- **`pm-roadmap.html`**: A visually interactive timeline illustrating the progressive upskilling sequences.
- **`resume.html` & `resume-all-regions.html`**: Highly stylized internet-native versions of my Resume intended for immediate URL dropping inside LinkedIn messages.
- **`linkedin-thumbnail.html`**: Formatted metadata rendering optimized open-graph sharing attributes.

## Usage Strategy
These HTML documents rely on the shared `../assets/` resources to render imagery. The focus here is rapid, visually striking content that verifies competency outside traditional PDF limitations.
